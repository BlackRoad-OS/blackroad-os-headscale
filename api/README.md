# BlackRoad headscale Enterprise API

## Overview

Proprietary API layer for BlackRoad headscale Enterprise.

## Features

- **Authentication**: JWT, OAuth 2.0, SSO
- **Rate Limiting**: Tier-based limits
- **Analytics**: Usage tracking and billing
- **Webhooks**: Event-driven integrations

## Endpoints

```
POST /api/auth/login
GET  /api/headscale/*
POST /api/headscale/*
```

## Integration

Connects to upstream headscale via standard APIs.
